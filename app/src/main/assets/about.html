ABOUT GPSSAMPLE:

GPSSample is an Android-based mobile application (version 8+) to facilitate enumeration and mapping, rapid statistical sampling, and navigation back to selected locations when conducting household surveys and rapid assessments. GPSSample is designed for use in the field, including areas with limited internet and/or mobile network coverage. The app features the following tiered user roles:

  • Admin sets up the configuration for an area, study questions, and defines the sampling design
  • Supervisor defines teams, manages field work, selects the sample, and reviews data
  • Enumerator maps and lists households/locations, collects points of interest, and syncs data with supervisor
  • Data Collector navigates back to the selected sample and completes the longer survey (ex. ODK)

GPSSample uses Mapbox offline map tiles. NOTE: each device will need to have maps downloaded before going to the field.

GPSSample uses a third-party survey tool, e.g., ODK. NOTE: each device will need to have the survey tool application downloaded before going to the field.

After a survey is completed, the GPSSample decoder can be used to decrypt the GPSSample json files for analysis. The GPSSample decoder tool and Technical Documentation are available on the GPSSample GitHub Page.
