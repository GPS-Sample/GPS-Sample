About GPSSample

GPSSample is an Android-based mobile application (version 8+) to facilitate enumeration and mapping, rapid statistical sampling, and navigation back to selected locations when conducting household surveys and rapid assessments. GPSSample is designed for the field, including areas with limited internet and/or cellular service. The app features the following tiered user roles:

  • Admin sets up the configuration for an area, study questions, and defines sampling design
  • Supervisor defines teams, manages field work, selects sample, reviews data
  • Enumerator maps and lists households, collects points of interest, syncs data with supervisor
  • Data Collector navigates back to the selected sample and completes the longer survey (ex. ODK)

 

GPSSample uses Mapbox offline map tiles. NOTE: each device will need to have maps downloaded before going to the field. After a survey, the GPSSample decoder can be used to decrypt the GPSSample json files for analysis.
